Introducción HTML


Este es un curso que pretende indroducir al lector en el mundo del HTML
No se requiere conocimiento precios, se empezara desde cero.
No hay que saber nada de programación para utilizar HTML, basta
aprender un par de cosas sencillas para empezar a crear.


Es muy facil aprender HTML. Tan solo necesitas un editor de texto, un
curso de HTML y un navegador web (como netscape o internet explorer),
ni siquiera necesitas estar conectado a Internet.
